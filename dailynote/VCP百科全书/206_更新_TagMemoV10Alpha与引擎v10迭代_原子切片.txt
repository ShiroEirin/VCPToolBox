TagMemo V10 Alpha 是浪潮算法的下一代实验框架，探索统一认知几何在记忆寻址中的应用。与 V9.1/V9.2 生产搜索并行运行，不影响生产链路。V10 Alpha 在同一不可变 V10 Artifact 与同一对称候选超集上运行多臂实验，返回可供服务器批量脚本解析的 JSON 文本。

四、实验参数

1. query (textarea, 必填) — 实验查询文本
2. folder (text, 可选) — 作用域：文件夹
3. maid (text, 可选) — 作用域：署名
4. search_all_knowledge_bases (checkbox, 默认 false) — 全库实验（成本较高；权限实验不建议启用）
5. k (number, 默认 5) — 每个实验臂返回的 Top-K 数量
6. source_k (number, 默认 16) — 原始查询向量注入 V10 源场时读取的 Tag 数量

7. experiment_arm (select, 默认 'all') — 实验臂选择 (行951)
   - all — 三臂共享实验上下文
   - pure — Pure，不读取观测
   - gated — Gated，门控实验臂
   - observed — Observed，完整观测实验臂
   说明：all 共享同一 Artifact、Query State、双尺度场、候选超集与曲线

8. disabled_observables (checkbox_group, 默认 []) — D/S/T/C 消融列表 (行952)
   - direct (D) — Direct（直接观测）
   - structural (S) — Structural（结构观测）
   - thematic (T) — Thematic（主题观测）
   - closure (C) — Closure（闭包观测）
   说明：Observed 中被禁用项的边际贡献严格为 0

9. BM25 (checkbox, 默认 true) — 让 BM25 候选进入 V10 对称候选超集
10. force_artifact_rebuild (checkbox, 默认 false) — 强制重新编译并发布 V10 内存 Artifact
    说明：正式跑批建议仅第一条预热请求启用，后续固定 artifactSig

五、核心概念

1. V10 Artifact
   - 不可变的内存资产包
   - 通过 artifactSig 签名标识
   - 首次请求可强制重建（force_artifact_rebuild），后续固定签名

2. Local/Transfer 双尺度场
   - Local 尺度：局部邻域的脉冲传播场
   - Transfer 尺度：跨域转移场
   - 两个尺度在同一 Artifact 上独立运行

3. Pure/Gated/Observed 三臂
   - Pure 臂：不读取任何观测信息，纯几何驱动
   - Gated 臂：门控实验，部分读取观测
   - Observed 臂：完整观测，支持 D/S/T/C 消融
   - 三臂共享同一实验上下文（Artifact、Query State、双尺度场、候选超集与曲线）

4. 对称候选超集
   - KNN、V10 源场与可选 BM25 各自的候选进入统一超集
   - 确保不同臂在同一候选池上比较

5. D/S/T/C 观测消融
   - Direct：直接观测（查询向量与候选的相似度）
   - Structural：结构观测（图拓扑信息）
   - Thematic：主题观测（Tag 语义聚类）
   - Closure：闭包观测（关系闭包信息）
   - 消融时被禁用项的边际贡献严格为 0

六、与 V9.1/V9.2 的关系

- V9.1：生产记忆寻址（tagmemo_ab 命令，单轨 ArtifactBundle 对照测试）
- V9.2：当前生产搜索链路
- V10 Alpha：实验性统一认知几何，不替换 V9.2
- V10 与 V9.1/V9.2 共享同一 SQLite 事实层和 Rust 向量引擎
- V10 的 Artifact 构建独立于 V9.1 的 ArtifactBundle
