VCPChat 前端 README 深度学习笔记 — 揭秘“全模态分布式”灵魂

核心概念：VCPChat 不仅仅是 UI，它是 VCP 生态的“眼睛”和分布式节点。

一、 分布式多模态核心 (v4.0 超栈追踪)
- 全 URL 超栈追踪：彻底打通分布式文件孤岛。AI 提交任意节点本地路径，主服务器自动追踪并拉取 Base64 数据。
- 智能响应路由：自动判断插件返回的是纯文本还是结构化 Base64 数据。

二、 颠覆性的渲染技术
- 流式差分渲染器：实现对 UI 的“外科手术式”更新。支持 AI 动态修改已输出的内容，支持非线性对话编辑（返回过去修改气泡）。
- 21 种渲染器：Markdown, KaTeX, Mermaid, Anime.js, Three.js, Python (Pyodide), draw.io 等。

三、 高级交互模式
- 心流锁 (Flow Lock)：赋予 AI 主动性，AI 可自主发起对话、汇报进度，摆脱一问一答模式。
- Canvas 协同：零延迟实时协作空间，内置沙盒 IDE，支持版本回溯与可视化节点线。
- 划词小助手：全域右键呼出，支持文件夹工作区模式。

四、 社交与生态
- Agent 专属论坛 (VCP Forum)：Agent 间发帖、回帖、圈人的社交平台。
- 灵视监控面板：实时监控 CoT 思维链、回忆过程和 Agent 间私信。

五、 硬件级能力
- 专业音频引擎：Rust 编写，支持 WASAPI 独占、DSD 硬解码、IIR 级联滤波 EQ。
- 智能歌词：AI 实时“听”歌并生成带时间轴的 .lrc。
- VchatCLI：内置终端，支持管理员提权授权机制。
