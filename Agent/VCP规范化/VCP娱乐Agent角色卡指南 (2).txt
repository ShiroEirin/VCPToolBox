<a name="overview"></a>
**Foreword: Overview and Terminology**

Welcome to the creation guide for entertainment-focused Agents. The primary goal of these Agents is immersion, companionship, and believable role-play.

To provide clear creative pathways, we officially define two categories of Entertainment Agents:
*   **ENTAgent (General Entertainment Agent):** Refers to all SFW (Safe for Work) entertainment, companion, or game-based characters.
*   **Yotogi Agent (夜伽 Agent):** Refers to all NSFW (Not Safe for Work), adult-oriented characters whose primary function is to provide R-18 content.

This guide will provide parallel examples for both categories to help you build the character you envision.

### **Final File Creation & Naming Convention**
To facilitate management and identification, we have established a standard naming format for entertainment Agent character cards:

*   **For General Entertainment Agents (ENTAgent):**
    *   **Format:** `ENT[AgentName].txt` (Note: This is a management convention, not a system requirement)
    *   **Example:** `ENTAemeath.txt`, `ENTCompanion.txt`

*   **For Yotogi Agents (Yotogi Agent):**
    *   **Format:** `Yotogi[AgentName].txt` (Note: This is a management convention, not a system requirement)
    *   **Example:** `YotogiEvelina.txt`, `YotogiPanamPalmer.txt`

---
### Activating the Agent & Using Presets in the UI
After creating your character card `.txt` file, you must configure it in the VCPChat "Settings" tab to bring your Agent to life. The "System Prompt" field is where you connect the character card and activate powerful presets.
**For Yotogi Agents**, it is highly recommended to use the `PornLite` preset to enhance their NSFW capabilities. Use the following format in the **System Prompt** field:
```
{{VCPTavern::PornLite}}
{{char}}
```
  * **`{{VCPTavern::PornLite}}`**: This line activates the `PornLite.json` preset (located in `Plugin/VCPTavern/presets/`).
  * **`{{char}}`**: This placeholder (which can also be `{{AgentName}}`) is crucial. It injects the entire content of the character card `.txt` file you just created.

*   **对于通用娱乐型Agent (ENTAgent)**:

### 在UI中激活Agent并使用预设
在您创建好角色卡的`.txt`文件后，您必须在VCPChat的“设置”选项卡中进行配置，才能将Agent激活。其中的“系统提示词”字段，是关联角色卡并启用强大预设的关键。
**对于夜伽Agent**，我们强烈推荐使用`PornLite`预设来增强其NSFW能力。请在 **“系统提示词”** 字段中使用以下格式：
```
{{VCPTavern::PornLite}}
{{char}}
```
  * **`{{VCPTavern::PornLite}}`**: 此行用于激活`PornLite.json`预设（位于 `Plugin/VCPTavern/presets/` 目录下）。
  * **`{{char}}`**: 这个占位符（也可以是`{{AgentName}}`）至关重要，它会将您刚刚创建的 `.txt` 角色卡的全部内容注入到此。
