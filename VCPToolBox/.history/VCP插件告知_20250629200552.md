# VCPToolBox 可配置插件指南

为了让您充分利用 VCPToolBox 的能力，了解所有可配置的插件及其功能是至关重要的一步。

此文档将所有可配置插件分为三类：**需要API密钥**、**需要连接信息** 和 **可调整功能**。

---

### 第一类：需要 API 密钥或认证信息

这些插件连接到外部付费或需要账户的服务。不提供正确的密钥，它们将无法工作。

1.  **`TavilySearch`** (网络搜索)
    *   **功能**: 赋予 AI 强大的网络搜索能力。
    *   **需要配置**: 在**根目录 `config.env`** 中找到 `TavilyKey` 并填入您的 Tavily API 密钥。

2.  **`BilibiliFetch`** (B站内容获取)
    *   **功能**: 让 AI 能够“看”B站视频，理解其内容。
    *   **需要配置**: 在**根目录 `config.env`** 中找到 `BILIBILI_COOKIE` 并填入您登录B站后的 Cookie 值。

3.  **`FluxGen`** (文生图)
    *   **功能**: 使用 SiliconFlow API 生成高质量图片。
    *   **需要配置**: 在 `Plugin/FluxGen/` 目录下创建 `config.env` 文件，并填入 `SILICONFLOW_API_KEY`。

4.  **`DoubaoGen`** (豆包文生图)
    *   **功能**: 使用火山方舟的豆包模型生成图片。
    *   **需要配置**: 在 `Plugin/DoubaoGen/` 目录下创建 `config.env` 文件，并填入 `VOLCENGINE_API_KEY`。

5.  **`NovelAIGen`** (动漫风格文生图)
    *   **功能**: 使用 NovelAI API 生成动漫风格的图片。
    *   **需要配置**: 在 `Plugin/NovelAIGen/` 目录下创建 `config.env` 文件，并填入 `NOVELAI_API_KEY`。

6.  **`SunoGen`** (AI音乐生成)
    *   **功能**: 使用 Suno API 生成原创歌曲。
    *   **需要配置**: 在 `Plugin/SunoGen/` 目录下创建 `config.env` 文件，并填入您的 `SunoKey` 和对应的 `SunoApiBaseUrl`。

7.  **`VideoGenerator`** (文生/图生视频)
    *   **功能**: 使用 Wan2.1 API 生成视频。
    *   **需要配置**: 在 `Plugin/VideoGenerator/` 目录下创建 `config.env` 文件，并填入您的 `SILICONFLOW_API_KEY`。

---

### 第二类：需要连接到您的其他服务

这些插件需要知道如何连接到您自己部署的其他服务。

1.  **`FRPSInfoProvider`** (内网穿透信息)
    *   **功能**: 让 AI 了解您 FRPS 服务器上的设备连接状态。
    *   **需要配置**: 如果您使用 FRPS，请在 `Plugin/FRPSInfoProvider/` 目录下创建 `config.env` 文件，并填入您的 FRPS 服务器地址、用户名和密码。

2.  **`VCPLog`** (日志推送)
    *   **功能**: 可以将 VCP 的日志通过 Gotify 推送到您的手机等设备上。
    *   **需要配置**: 如果您使用 Gotify，请在 `Plugin/VCPLog/` 目录下创建 `config.env` 文件，并填入您的 Gotify 服务地址和应用 Token。

---

### 第三类：可调整功能参数

这些插件开箱即用，但您可以通过配置来微调它们的功能。

1.  **`ArxivDailyPapers`** (学术论文获取)
    *   **功能**: 定期从 Arxiv.org 拉取您感兴趣领域的最新论文。
    *   **可配置**: 您可以在 `Plugin/ArxivDailyPapers/` 目录下创建 `config.env` 文件，修改 `ARXIV_SEARCH_TERMS` 来定义您关心的**搜索关键词**，例如 `"Large Language Models"`。

2.  **`CrossRefDailyPapers`** (已发表论文获取)
    *   **功能**: 从 CrossRef 拉取已发表的学术论文元数据。
    *   **可配置**: 类似 Arxiv 插件，您可以修改 `CROSSREF_QUERY_BIBLIOGRAPHIC` 来定义搜索的关键词。

3.  **`AgentAssistant`** (多 Agent 协同)
    *   **功能**: 这是 VCP 的核心功能之一，允许您定义多个拥有不同角色、不同模型的 AI 助手，并让它们协同工作。
    *   **可配置**: 您可以在 `Plugin/AgentAssistant/` 目录下创建 `config.env` 文件，定义您自己的 Agent 团队。例如，您可以定义一个“代码助手”和一个“创意助手”，它们使用不同的模型和系统提示词。

---

### 总结与建议

*   **无配置也可运行的插件**: `SciCalculator` (科学计算器) 不需要任何配置。
*   **优先配置**: 建议您优先关注**第一类**中您会用到的插件，为它们提供 API 密钥。
*   **按需配置**: **第二类**和**第三类**插件，您可以根据自己的实际使用场景来决定是否需要进行配置。
