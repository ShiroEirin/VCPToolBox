# VCPToolBox 插件配置指南 (完整版)

此文档根据项目官方 `README.md` 文件整理，旨在提供一份完整、准确的插件功能及配置说明。

## 核心能力分类

### 一、 环境感知与信息获取

这类插件让 AI 能够感知外部世界的实时信息。

1.  **`WeatherReporter`** (天气预报)
    *   **功能**: 提供实时天气信息，包括预警、小时详情和多日预报。
    *   **配置**: 需要在**根目录 `config.env`** 中提供和风天气等服务商的 `WeatherKey` 和 `WeatherUrl`。

2.  **`TavilySearch`** (网络搜索)
    *   **功能**: 集成 Tavily API，赋予 AI 强大的网络搜索能力。
    *   **配置**: 需要在**根目录 `config.env`** 中提供 `TavilyKey`。

3.  **`UrlFetch`** (网页抓取)
    *   **功能**: 基础的网页内容抓取工具。
    *   **配置**: 无需额外配置。

### 二、 学术文献获取

这类插件专注于从学术数据库获取最新研究成果。

1.  **`ArxivDailyPapers`** (Arxiv 论文)
    *   **功能**: 从 Arxiv.org 获取您指定领域的每日最新论文。
    *   **配置 (可选)**: 可在 `Plugin/ArxivDailyPapers/` 目录下创建 `config.env` 文件，通过 `ARXIV_SEARCH_TERMS` 自定义搜索的关键词。

2.  **`CrossRefDailyPapers`** (CrossRef 论文)
    *   **功能**: 从 CrossRef 获取已发表的学术论文元数据。
    *   **配置 (可选)**: 可在 `Plugin/CrossRefDailyPapers/` 目录下创建 `config.env` 文件，通过 `CROSSREF_QUERY_BIBLIOGRAPHIC` 自定义搜索的关键词。

### 三、 多模态处理与内容生成

这类插件赋予 AI 理解和创造图片、视频、音乐等多媒体内容的能力。

1.  **`ImageProcessor`** (图像理解)
    *   **功能**: 自动识别用户消息中的图片，并将其内容转换为文字描述供 AI 理解。
    *   **配置**: 在**根目录 `config.env`** 中配置 `ImageModel` (指定用于识别的视觉模型)等相关参数。

2.  **`FluxGen`** (文生图)
    *   **功能**: 使用 SiliconFlow API 的 FLUX 模型生成高质量图片。
    *   **配置**: 需要在 `Plugin/FluxGen/` 目录下创建 `config.env` 文件，并填入 `SILICONFLOW_API_KEY`。

3.  **`Wan2.1VideoGen`** (视频生成)
    *   **功能**: 使用 SiliconFlow API 的 Wan2.1 模型进行文本到视频或图像到视频的生成。这是一个**异步插件**，任务在后台执行。
    *   **配置**: 需要在 `Plugin/VideoGenerator/` 目录下创建 `config.env` 文件，并填入 `SILICONFLOW_API_KEY`。

4.  **`SunoGen`** (音乐生成)
    *   **功能**: 使用 Suno API 生成原创歌曲。
    *   **配置**: 需要在 `Plugin/SunoGen/` 目录下创建 `config.env` 文件，并填入 `SunoKey` 和 `SunoApiBaseUrl`。

5.  **`DoubaoGen`** (豆包文生图)
    *   **功能**: 使用火山方舟的豆包模型生成图片。
    *   **配置**: 需要在 `Plugin/DoubaoGen/` 目录下创建 `config.env` 文件，并填入 `VOLCENGINE_API_KEY`。

### 四、 核心记忆与知识管理

这是 VCP 的灵魂，让 AI 能够学习、记忆和进化。

1.  **`DailyNoteGet`** (日记读取)
    *   **功能**: 定期读取所有角色的日记，并将其注入到 AI 的上下文中。
    *   **配置**: 无需额外配置。

2.  **`DailyNoteWrite`** (日记写入)
    *   **功能**: 允许 AI 主动将学习到的知识、经验写入结构化的日记。
    *   **配置**: 无需额外配置。

3.  **`DailyNoteManager`** (日记管理)
    *   **功能**: 允许 AI 主动整理、优化、检查和共享自己的知识库。
    *   **配置**: 无需额外配置。

4.  **`DailyNoteEditor`** (日记编辑)
    *   **功能**: 允许 AI 自我修正和完善知识库中的内容。
    *   **配置**: 无需额外配置。

### 五、 Agent 协同与系统增强

这类插件增强了 VCP 的系统能力和多 Agent 协作能力。

1.  **`AgentAssistant`** (Agent 间协作)
    *   **功能**: 核心插件，允许 AI 之间进行即时或**定时**的通讯、任务分发和协作。
    *   **配置 (可选)**: 可在 `Plugin/AgentAssistant/` 目录下创建 `config.env` 文件，详细定义每个 Agent 的模型、角色和能力。

2.  **`AgentMessage`** (对用户发消息)
    *   **功能**: 允许 AI 通过 WebSocket 主动向用户前端发送格式化消息。
    *   **配置**: 无需额外配置。

3.  **`VCPLog`** (日志服务)
    *   **功能**: 记录 VCP 工具调用的日志，并可通过 WebSocket 推送。
    *   **配置 (可选)**: 可在 `Plugin/VCPLog/` 目录下创建 `config.env` 文件，配置通过 Gotify 将日志推送到您的设备。

4.  **`EmojiListGenerator`** (表情包列表生成)
    *   **功能**: 自动扫描 `image/` 目录下的表情包文件夹，生成列表供 AI 使用。
    *   **配置**: 无需额外配置。

5.  **`ImageServer`** (图床服务)
    *   **功能**: 提供一个带密钥认证的静态图床服务，用于访问生成的图片等。
    *   **配置**: 在**根目录 `config.env`** 中配置 `Image_Key`。

### 六、 专业工具集成

1.  **`SciCalculator`** (科学计算器)
    *   **功能**: 提供强大的科学计算能力，支持从基础运算到微积分的复杂数学表达式。
    *   **配置**: 无需额外配置。
